xpassgen
========

Generates XKCD-style passphrases with a 10k ESL wordlist and SecureRandom.


Sample output
-------------

    Word list has 10231 words, or about 13.3 bits per word
    Using SecureRandom: SUN version 1.7 SHA1PRNG
